# 🌱 Agrinho 2026

## 📚 Tema

Recuperação de Áreas Degradadas para Nova Produção Agrícola.

---

## 🎯 Objetivo

Este projeto foi desenvolvido para conscientizar sobre a importância da recuperação ambiental e do uso sustentável do solo.

O site apresenta:
- causas da degradação;
- soluções sustentáveis;
- benefícios ambientais;
- interação com o usuário;
- recursos modernos de acessibilidade.

---

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📱 Recursos do Projeto

✅ Responsividade  
✅ Modo escuro  
✅ Contador interativo  
✅ Hover e animações  
✅ Flexbox  
✅ Media Queries  
✅ Manipulação do DOM  
✅ Alteração dinâmica da fonte  

---

## 🖼️ Mídias Utilizadas

As imagens podem ser:
- próprias;
- editadas no Canva;
- modificadas no Paint;
- produzidas no Piskel.

---

## 🚀 Como Executar

1. Baixe os arquivos.
2. Coloque as imagens na pasta `/img`.
3. Abra o arquivo `index.html` no navegador.

---

## 👨‍💻 Organização

O projeto possui:
- HTML separado;
- CSS separado;
- JavaScript separado;
- pasta exclusiva para imagens.

---

## 🌎 Mensagem Final

Preservar o solo hoje significa garantir alimento e qualidade de vida para o futuro.
